# Project Hiawatha
A Civilization knock-off written in C++ with Qt 5.7.0 by Paul Jackemeyer, Alex Portolese, and Sam Stein

##Contents:

###Main Features

1. Random Map Generation
2. AI
3. Civs
4. Tech Tree
5. Strategic and Luxury Resources
6. War
